# Landing Page - Giórgia Ghinato

Projeto React + Vite com Framer Motion para uma landing page premium executiva.

## Como rodar

```bash
npm install
npm run dev
```

## Como gerar build

```bash
npm run build
```

## Observações

- Os dados de telefone e email em `src/sections/ContactSection.jsx` estão como exemplo e podem ser substituídos facilmente.
- Caso queira adicionar foto da Giórgia, o lugar mais elegante para inserir é no `HeroSection.jsx`, dentro do bloco `.hero-panel`.
